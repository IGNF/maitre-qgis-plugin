# Présentation du plugin MAITRE

## 

Ce plugin permet de gérer l'intégration d'autres plugins IGN dans un menu et dans des barres d'outils.

Pour être pris en compte, ces plugins doivent être préfixés par « IGN\_ ».

Ce préfixe sert de filtre parmi tous les plugins existants.



## 

L'utilisation du plugin est décrite dans son [manuel utilisateur](plugin_maitre.pdf).
